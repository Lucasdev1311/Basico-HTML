Como tudo começou

Pra falar sobre o inicio da web devemos ir um pouco mais ao passado, dando ínicio juntamente a guerra fria, acontece que a necessidade de ser superior sempre esteve atrelada aos países isso pode ser ruim pelo fato de sempre haver guerras no meios das revoluções, mas por outro lado o avanço tecnológico por meio dessas revoluções foram algo extremamente incrível.

Um exemplo que temos claramente sobre grandes avanços, que não podemos deixar de ressaltar é como os computadores evoluíram com o tempo, seu propósito no início era simplesmente fazer cálculos voltado para a guerra e comunicação, como tinhamos o telégrafo que enviava código morsa para ser comunicar

Podemos citar alguns computadores da época:

  • MARK 1, um dos primeiros computadores, 1946.
  • COLOSSUS, criado pela alemanha, 1946.
  • ENIAC, criado pelos EUA.

Obeservações

  • Segunda geração 2°

    • Com o passar do tempo os computadores foram evoluindo e se tornando cada vez mais eficiente, de 1959 à 1965 os computadores evoluíram de válvulas para transitores fazendo com que assim reduzissem de tamanho isso marca a segunda geração.

  • Terceira greação 3°

    • De 1965 à 1970 temos terceira geração temos o surgimento de chips, circuitos integrados, transitores, capacitores etc.

  • Quarta geração 4°

    • A quarta geração parte de 1971 até os dias de hoje,com as devidas evoluções, as melhorias e desenvolvimento.

O que são clientes?

De uma maneira bem resumida podemos dizer que clientes são os dispositivos que usamos para nos conectar a outro lugar. Os clientes consomem informação através da internet que esta contida em servidores, toda essa conexão é feita através de browsers conhecido como navegadores.

O que são servers?

Um pouco de tags HTML

Tags HTML nada mais é que marcadores utilizados no texto, que vão determinar parágrafos, títulos etc.

Logo abaixo vamos dar alguns exemplos de tags.

  • Tag (p): É utilizada para maracações de parágrafos.
  • Tag (font): Era utilizada nas versões antiga do HTML, não suportada no HTML5, toda modificação deve ser feita através do CSS.
  • Tag (h1 até h6): São tags geralmente usadas para título.
  • Tag (mark): Utilizada para marcar texto.
  • Tag (del): Usada para definir um texto deletado de um documento.
  • Tag (abbr): Usada para definir abreviações ou acronimos.
  • Tag (a): É utilizada para links de navegação seja na mesma página ou ir para outras páginas.
  • Tag (u): Utilizada para sublinhar
  • Tag (blockquote): Usada para citações, frases de outro autor.
  • Tag (small): Usada para definir copy-right e alguns comentários.
  • Tag (i): Usada para deixar o texto em itálico.
  • Tag (strong): Usada para deixar o texto em negrito.
  • Tag (ol): Utilizada para fazer listas ordenadas.
  • Tag (ul): Usada para listas desordenadas.
  • Tag (li): Utilizada para indicar uma lista.
  • Tag (hr): É usada geralmente em de forma horizontal, usando uma linha para separar determinado texto, ela cria uma linha na horizontal.
  • Tag (sub): Usada para definir subscript, como por exemplo uma fórmula química na mesma linha.
  • Tag (sup): Utilizada para superscript, ou footnote(notas de rodapé).
